Challenges


Create a web application that continuously collects and visualizes all of the devices connected on the simulated home network. To help simplify the challenge, all of the raw data collected will be provided. Consider what a useful dashboard might look like with the given data. Create at least one visualization that draws a representation of the home network. Then try to provide answers to questions people might have about their home network; like, what’s causing my internet so slow?

Where to start:
Go to http://<hostname>/devices/ for a list of devices’ URLs

Scoring categories:
1. Demo Presentation
2. Appearance
3. Accuracy of Data Presented
4. Innovation
5. Code Quality
6. Usefulness


Using a Content Management System (Wordpress, Drupal or Joomla) and HTML/CSS/PHP, the objective is to implement a simple resume processing system.

Requirements:
1. Job seekers register on website. Registration includes name, email address, password selection and facility to upload PDF file into the content management system (CMS). All resumes are in PDF format; all other formats are to be rejected.
2. HR account with initial default password. Once logged in HR can modify password.
3. HR uses a search feature (textbox) where the HR can filter the resumes based on matching key words/phrases to be matched against all resumes.
4. Result of analysis is a web page ranking resumes from most matches to least.
5. Resumes are to be hyperlinked so that HR can review any resume
6. HR sends results to job seekers through the CMS directly
